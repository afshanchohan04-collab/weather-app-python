# 🌦 Weather Web App (Flask + API)

A simple and interactive weather web application built using Python, Flask, and OpenWeather API.  
This app allows users to search for any city and view real-time weather information.

---

## 🚀 Live Demo
👉 https://yourusername.pythonanywhere.com

---

## 📌 Features

- 🌍 Search weather by city name
- 🌡 Displays temperature in Celsius
- ☁ Shows weather condition (clear, cloudy, rain, etc.)
- 💧 Displays humidity
- 🔁 Handles invalid city input with error messages
- 🎨 Clean and simple user interface

---

## 🛠️ Technologies Used

- Python
- Flask (Web Framework)
- HTML & CSS
- OpenWeather API
- Requests Library

---

## 🧠 How It Works

1. User enters a city name
2. Flask backend receives input
3. API request is sent to OpenWeather
4. JSON data is processed
5. Weather details are displayed on the web page
